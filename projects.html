01

Linux Server Health Monitor

A Bash + Python tool that continuously monitors a Linux server's vital signs — CPU usage, memory consumption, disk space, and running processes. When any metric crosses a defined threshold, it logs the event and triggers an alert so you catch problems before they become outages.

Real-time CPU, memory & disk monitoring
Configurable alert thresholds
Automated logging with timestamps
Runs as a cron job — fully automated
Bash Python cron Linux syslog
GitHub repo Live demo
